# Raw Dataset

Folder ini berisi dataset mentah (raw dataset) yang digunakan dalam penelitian.

## Struktur Folder

```text
raw/
├── 1778217759.csv
└── raw.md
```

## Deskripsi File

| File | Keterangan |
|------|------------|
| `1778217759.csv` | Dataset mentah yang digunakan sebagai sumber data utama penelitian. |

## Deskripsi Atribut

| Atribut | Tipe Data | Peran | Deskripsi |
|----------|-----------|-------|-----------|
| Komoditas | Kategorikal | Input | Nama komoditas pangan yang diamati. |
| Tahun | Integer | Input | Tahun pencatatan harga. |
| Bulan | Kategorikal | Input | Bulan pencatatan harga. |
| Harga | Numerik | Target | Nilai rata-rata harga komoditas pangan pada tingkat konsumen. |

## Informasi Dataset

- **Nama Dataset:** Rata-rata Harga Pangan Bulanan Tingkat Konsumen Nasional (Update Harga April 2026)
- **Periode Data:** 2021–2026
- **Format:** CSV
- **Jumlah Data:** 1.227 baris dan 4 kolom

## Sumber Dataset

Dataset diperoleh dari Portal Satu Data Indonesia:

https://data.go.id/dataset/dataset/rata-rata-harga-pangan-bulanan-tingkat-konsumen-nasional-update-harga-april-2026
